Module description
==================

.. topic:: Overview

    CN-Wheat is a model simulating the distribution
    of carbon and nitrogen metabolites into wheat plants described at organ level.
    Synthesis, degradation and transport of various C-N metabolites are
    simulated for each individual shoot organs and a single root compartment.
